# 列表

1. 列表中每个值都有对应的位置值，称之为索引，第一个索引是 0，第二个索引是 1，依此类推(即从0开始，到列表长度减1)
2. 列表的索引值可以是负数，从-1开始，到-n，依次递减
3. 列表可以进行截取、组合等操作

:::code-group

```python:line-numbers
a = [1, 2, 3, 4, 5]
print(a[0])
print(a[-1])
print(a[1:3])
print(a[::2])
print(a[::-1])
```

```输出结果
1
5
[2, 3]
[1, 3, 5]
[5, 4, 3, 2, 1]
```

:::
